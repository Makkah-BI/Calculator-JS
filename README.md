# Calculator

A fully functional calculator web app for basic arithmetic operations.

## Features
- Basic operations: +, -, ×, ÷
- Decimal point support
- Clear (C) and backspace (⌫) buttons
- Equals (=) to compute result
- Keyboard support (numbers, operators, Enter, Backspace, Escape)
- Error handling (division by zero, invalid expressions)
- Responsive button grid

## Technologies
- HTML5
- CSS3 (Grid layout, shadows, rounded corners)
- JavaScript (event listeners, safe evaluation using Function and custom logic)
